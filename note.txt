# AVERTISSEMENT DE SÉCURITÉ

Ce fichier contenait une copie du fichier .env avec des credentials sensibles.
Il a été nettoyé pour des raisons de sécurité.

ACTIONS REQUISES :
1. Ajouter "note.txt" dans le .gitignore
2. Révoquer et régénérer les clés Cloudinary sur https://cloudinary.com/console
3. Révoquer et régénérer les credentials Mailtrap sur https://mailtrap.io
4. Ne jamais stocker de credentials en clair dans des fichiers versionnés

Le fichier .env (déjà dans .gitignore) est la seule source de configuration autorisée.
